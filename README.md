# Neha Reddy K - Personal Portfolio Website

A premium, highly interactive personal portfolio website showcasing B.Tech Computer Science & Data Science credentials, internships, courses, and engineering projects.

## 🚀 Live Preview
The website runs a responsive single-page architecture featuring a customized Glassmorphism layout.

## ✨ Premium Features
- **Modern Design System**: Sleek default dark theme with a toggle to a polished light theme.
- **Interactive Particle Canvas**: Embedded canvas background that dynamically connects nodes and responds to theme color switches.
- **Typewriter Effect**: An automated looping header showcasing primary tech roles.
- **Category-Based Skills Filtering**: Custom tag classification allowing recruiters to filter by Web Dev, Data Science, Tools, and Soft Skills with entry animations.
- **Interactive Experience Timeline**: Elegant vertical cards describing internship achievements at Prodigy Infotech and Unify Labs.
- **Professional Print Styles (`@media print`)**: Accessing the "Resume" download button triggers a custom stylesheet print preset, formatting the entire layout into a clean, traditional 2-page print layout.
- **Contact Clipboard Actions**: Clickable buttons copy fully formatted contact strings directly to clipboards.
- **Client-Side Form Validation**: Structured input checks with real-time field status and submission animations.

## 🛠️ Built With
- **HTML5**: Semantic document layout
- **Vanilla CSS3**: Layout grids, flex structures, theme configurations, keyframe animations, and media queries
- **JavaScript (ES6+)**: Canvas animations, scroll triggers, event filters, state management, and clipboard utilities
- **FontAwesome CDN**: Tech icons and graphic assets
- **Google Fonts (Outfit & Inter)**: Premium typography

## 💻 Setup and Local Run
To run the project locally, clone the repository and run any local server from the root directory:
```bash
# Clone the repository
git clone https://github.com/nehaa225/PERSONAL-PORTFOLIO.git

# Serve using http-server (npm)
npx http-server -p 8080
```
Then navigate to `http://127.0.0.1:8080` in your web browser.
